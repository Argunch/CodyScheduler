**CodyScheduler** is a web-based scheduling application I built to manage my own coding and robotics lessons. It allows me to create and manage both one-time and recurring events, assign colors, set durations, and control access with a role-based system.

> 🚀 Built with **Django** (Python) + **Vanilla JS** + **HTML/CSS**.
> 📦 Currently used in production at [scheduler.mrrob.ru](https://scheduler.mrrob.ru/).

## ✨ Key Features

- **Calendar management**: Create, edit, move, and delete events.
- **Recurring events**: Supports both one‑time and repeating schedules.
- **Color coding**: Assign custom colors to different event types.
- **Access control**: Role‑based views (Admin vs. regular user).
- **Responsive design**: Works seamlessly on desktop and mobile.
